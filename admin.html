💳 SnapScan Payment Sync

After each match, the treasurer exports successful transactions from the SnapScan campaign and pastes the reference codes into the Payments tab of the Google Sheet (column B). Then click Sync to automatically mark all matching votes as paid.

How to set up the Payments sheet:
  1. Open the Google Sheet ↗
  2. Go to the Payments tab (auto-created on first sync)
  3. Column A: timestamp · Column B: reference code · Column C: amount · Column D: note
  4. Paste SnapScan export rows — only the reference column (B) matters for matching
  5. Click Sync below
