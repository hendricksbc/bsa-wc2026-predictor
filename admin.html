⚽ Match Results

Match results are fetched automatically from the football-data.org API every 2 hours via GitHub Actions. No manual entry is required. The leaderboard and raffle entries update automatically once results are available and payments are confirmed.

💳 SnapScan Payment Sync

Export successful transactions from the SnapScan WC2026 campaign and paste the reference codes into column B of the Payments tab in the Google Sheet. The system auto-syncs every hour, or click below to sync immediately.

Payments sheet format:
  1. Open the Google Sheet ↗ → Payments tab
  2. Column B = reference code (e.g. BSA-CYZQ) — this is the only column that matters for matching
  3. Paste SnapScan export rows — header row is already in place
